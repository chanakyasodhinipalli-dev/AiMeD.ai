# AIMeD.ai (GitHub Pages-ready)

This repo contains a static site in `/docs` (recommended GitHub Pages source).

## Quick deploy (GitHub Pages)
1. Push this repo to GitHub.
2. Settings → Pages → **Deploy from a branch**
3. Branch: `main` (or `master`) → Folder: `/docs`
4. Your site goes live on the GitHub Pages URL.

## Customize
- Primary copy & module cards: `docs/assets/content.json`
- Styling: `docs/assets/styles.css`
- Shared layout/behavior: `docs/assets/site.js`

## Notes
- Contact form is UI-only for now (connect to Formspree/Netlify/your backend later).
